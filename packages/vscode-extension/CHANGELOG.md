# Changelog

## 0.1.0

- Initial release for the Cascades VS Code extension.
- Added commands:
  - Cascades: Getting Started — onboarding walkthrough
  - Cascades: Set Session Cookie — authentication helper
  - Cascades: Open Dashboard — opens the web dashboard
  - Cascades: Open Workflow Builder — visual DAG editor
  - Cascades: List Workflows — browse catalog
  - Cascades: Run OSINT Intake — guided SpiderFoot/Maltego/etc collection
  - Cascades: Run Investigation — full investigation pipeline
  - Cascades: Verify Evidence — generate and verify cryptographic proofs
  - Cascades: Show Run Status — check execution results
  - Cascades: Open Documentation — quick section picker
  - Cascades: Insert Code Example — pasteable Python snippets
  - Cascades: About — version and links
- Error messages include documentation URLs for quick troubleshooting
- All commands authenticate via session cookie with guided setup
